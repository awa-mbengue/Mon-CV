# 📄 CV Interactif – Awa Mbengue

## 📌 Description

Ce projet est un **CV interactif** développé en **HTML5, CSS3 et JavaScript** dans le cadre d'un projet académique en Licence 2 Informatique.

L'objectif est de présenter mon parcours, mes compétences, mes expériences et mes projets à travers une interface moderne, responsive et interactive.

---

## ✨ Fonctionnalités

* Présentation personnelle
* Formation académique
* Compétences avec barres de progression animées
* Expériences professionnelles
* Projets réalisés
* Langues et centres d'intérêt
* Formulaire de contact avec validation JavaScript
* Mode sombre / mode clair
* Bouton "Retour en haut"
* Menu responsive pour mobile
* Design moderne et adaptatif

---

## 🛠️ Technologies utilisées

* HTML5
* CSS3
* JavaScript (ES6)
